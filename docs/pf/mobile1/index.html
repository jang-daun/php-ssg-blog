04. Character
편의점 인공위성 ‘위(oui)’
편위의 대표 캐릭터 이름은 위(oui)입니다. oui는 사전적 의미로 ‘yes’를 뜻하며 발음할 때 귀여운 느낌을 줍니다.
캐릭터의 이미지에 알맞게 귀여운 느낌과 긍정적인 의미를 담아 위(oui)라고 지었습니다.
위(oui)는 편의점 탐지 구슬로 편의점 정보를 탐색합니다.
편의점 탐지 구슬은 각 편의점 브랜드의 대표 컬러로 이루어져있습니다.
